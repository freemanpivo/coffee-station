## Getting Started

Currently building... üèóÔ∏è

### Prerequisites

- OpenJDK 17
- Terraform
- Docker
- Go

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/freemanpivo/coffee-station.git
   ```
2. Run Docker
   ```sh
   docker-compose up
   ```
